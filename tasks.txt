Техническое задание
Общее

В шапка отображается текущее количество заметок
Адаптив под мобильные устройства до 375px - на ваше усмотрение (не должно быть горизонтального скролла)
Когда заметок нет, отображается текст: “У вас ещё нет ни одной заметки. Заполните поля выше и создайте свою первую заметку!”
Избранное

В шапке заметки есть есть кнопка "Добавить в избранное". При нажатии на неё заметка добавляется в избранное. При повторном нажатии - удаляется из избранного
Иконка кнопки "Добавить в избранное" меняется в зависимости от того, добавлена ли заметка в избранное
При включении фильтра “Показать только избранные заметки” отображаются только заметки, добавленные в избранное
Удаление заметок

В шапке заметки есть кнопка удаления. При нажатии на неё заметка удаляется, общее количество заметок пересчитывается
При удалении заметки должно быть показано сообщение “Заметка удалена”. Сообщение скрывается через 3 секунды
Форма добавления заметки

В форме добавления заметки есть поля для ввода заголовка и текста заметки, а также выбор цвета заметки. Изначально в форме выбран жёлтый цвет заметок
Новые заметки добавляются в начало списка
При добавлении заметок должно быть показано информационное сообщение “Заметка добавлена”. Сообщение скрывается через 3 секунды
При отправке формы с незаполненными полями появляется сообщение “Заполните все поля”, добавление заметки не происходит. Цвет сообщения - красный
Максимальная длина заголовка заметки - 50 символов. При попытке добавить заметку с заголовком больше 50 символов, появляется сообщение “Максимальная длина заголовка - 50 символов”. Цвет сообщения - красный
При успешном добавлении заметки форма очищается, в противном случае - остаётся заполненной
